# Simple Todo App

A simple Todo app! This Web application allows users to add unique to-do's while giving them the optionional choice to set a due date for their todo task. This app implements JavaScript, focusing on OOP.

## Functionality

Functional/Validated forms to input a to-do task or remove it with the ability to implement due dates with live time zones.

## Technology

HTML/CSS, JavaScript, Git, and OOP
